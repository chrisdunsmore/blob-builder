# blob-builder
Find and Replace multiple Blobs at once with variables declared

Helps extract URLs from existing "messy" HTML and then set them as find/replace variables to be injected into a replace template.

Replace template is set up by saying <a href="{var1}">info</a> ... when exectuted whatever is stored in Var1 field gets inserted.  

Previous to that use the harvest/extracter to populate Vars
